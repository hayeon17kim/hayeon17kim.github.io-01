## 강의노트
비트캠프 엄진영 강사님의 수업을 들으며 정리하였습니다.



```java
    if (gender == 1) {  
      System.out.println("여성"); 
    } 
    
    if (gender == 2) {
      System.out.println("남성");
    }
// 비효율적인 코드

// 배타적인 조건일 경우 실행하지 않기 위해서 else 라는 문법
```

```java
    int age = 66;

    if (age >= 19) {  
      System.out.println("성인입니다."); 
    } 
    
    if (age >= 65) {
      System.out.println("노인연금 수령 대상자입니다.");
    }
```



자바에는 **else if** 문법이 없다. if와 else만이 있을 뿐이다.
