## int

- primitive 자료형
- 산술연산 가능
- null로 초기화 불가능



## Integer

- wrapper 클래스
- 산술연산 불가능 (Unboxing 필요)
- null값 처리가 용이해 SQL과 연동하기 좋음
- int를 클래스로 감싼 형태



## Wrapper 클래스

- Boxing: Primitive 자료형 => Wrapper 클래스
- Unboxing: Wrapper 클래스 => Primitive 자료형
